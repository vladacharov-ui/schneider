Политика конфиденциальности

Настоящая Политика конфиденциальности персональных данных (далее — Политика) действует в отношении всей информации, которую Администрация сайта (далее — Оператор) может получить о пользователе во время использования им сайта Schneider Weisse Brauhaus (далее — Сайт).

1. Общие положения

1.1. Использование пользователем Сайта означает согласие с настоящей Политикой конфиденциальности и условиями обработки персональных данных пользователя.

1.2. В случае несогласия с условиями Политики пользователь должен прекратить использование Сайта.

1.3. Настоящая Политика применяется только к данному Сайту. Оператор не контролирует и не несет ответственность за сайты третьих лиц, на которые пользователь может перейти по ссылкам, доступным на Сайте.

1.4. Оператор не проверяет достоверность персональных данных, предоставляемых пользователем Сайта.

2. Персональные данные, подлежащие обработке

Оператор осуществляет обработку следующих персональных данных Пользователя, предоставляемых при заполнении формы бронирования стола:

  • Имя (для идентификации Пользователя и обращения к нему);
  • Номер контактного телефона (для подтверждения бронирования и связи с клиентом);
  • Дата и время планируемого бронирования;
  • Количество гостей;
  • Комментарии к бронированию (любые дополнительные пожелания, оставленные добровольно).

3. Цели сбора персональных данных

Персональные данные пользователя Оператор может использовать в целях:

  • Идентификации пользователя для оформления резерва стола в ресторане Schneider Weisse Brauhaus;
  • Установления с пользователем обратной связи, включая направление уведомлений, телефонных звонков от администратора для подтверждения брони;
  • Предоставления пользователю эффективной клиентской и технической поддержки при возникновении проблем, связанных с использованием Сайта;
  • Направления рекламных, новостных и специальных предложений (только при наличии отдельного согласия пользователя).

4. Правовые основания обработки данных

4.1. Обработка персональных данных осуществляется в соответствии с требованиями Федерального закона № 152-ФЗ "О персональных данных".

4.2. Правовым основанием обработки является Согласие пользователя на обработку персональных данных, выражаемое путем проставления галочки (акцепта) в чекбоксе перед отправкой формы бронирования.

5. Порядок обработки и хранения персональных данных

5.1. Обработка персональных данных пользователя осуществляется без ограничения срока любым законным способом (сбор, запись, систематизация, накопление, хранение, уточнение, извлечение, использование, передача, блокирование, удаление).

5.2. Персональные данные пользователя хранятся на защищенных серверах и не передаются третьим лицам, за исключением случаев, связанных с исполнением действующего законодательства Российской Федерации.

6.1. Пользователь имеет право на получение информации, касающейся обработки его персональных данных, а также на их уточнение, блокирование или уничтожение.
